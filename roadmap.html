1. Introduction to Git

Git is a distributed version control system used to manage source code history and enable collaboration. Itâ€™s essential in every DevOps pipeline.

2. Installing Git


# Ubuntu
sudo apt update
sudo apt install git

# macOS
brew install git

# Configure Git
git config --global user.name "Your Name"
git config --global user.email "you@example.com"

3. Git Commands You Must Know


git init                  # Initialize repo
git clone       # Clone repository
git status                # View changes
git add .                 # Stage changes
git commit -m "message"   # Commit changes
git push origin main      # Push to remote

4. Branching & Merging


git checkout -b dev       # Create and switch to dev branch
git merge dev             # Merge into current branch
git branch -d dev         # Delete dev branch

5. Git in DevOps

  • Trigger Jenkins pipelines using GitHub Webhooks
  • GitOps: manage Infra-as-Code (Terraform, K8s) via Git
  • Follow branching strategies like Git Flow
